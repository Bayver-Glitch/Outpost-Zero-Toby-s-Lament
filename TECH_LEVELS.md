# OUTPOST ZERO - Tech Level & Progression System

## CORE CONCEPT

**Tech Level (1-5):**
- Determines what you can build, craft, and research
- Starts at Level 1 (Scrap/Stone Age)
- Progresses to Level 5 (Modern/High-Tech)
- Each level unlocks new buildings, recipes, and capabilities

---

## TECH LEVELS

## SKILL UNLOCKING SYSTEM

### Learning Methods

1. **Books & Magazines**
   - Find in scavenging runs
   - Each book unlocks a skill or improves existing
   - Condition matters (water-damaged = partial understanding)
   - Some books in foreign languages (need translator)

2. **Teaching**
   - Character with skill 30+ can teach
   - Teaching efficiency depends on Teacher skill
   - Requires student to have basic education (reading/writing)

3. **Practice**
   - Using skill = XP gain
   - Level up every 20 XP

4. **Writing Books**
   - Character skill 40+ can write basic guide
   - Skill 60+ can write advanced guide
   - Must have Education level 3+ (high school)
   - Adds new book to library
   - Other characters can learn from it

### Skill Book Requirements

| Skill | Book Needed | Education Required |
|-------|-------------|-------------------|
| Farming | Any farming book | Basic |
| Hunting | Hunting guide | None |
| Fishing | Fishing manual | None |
| First Aid | First aid book | Basic |
| Blacksmithing | Blacksmith guide | Reading |
| Medicine | Medical textbook | Grade 8+ |
| Mechanics | Mechanics guide | Grade 10+ |
| Electrical | Electrical basics | Grade 10+ |
| Programming | Computer science | College |

## CHARACTER EXPECTATIONS

### High-Tech Characters

Characters who have lived with higher tech have **expectations**:

| Character Background | Minimum Comfort Tolerance |
|--------------------|------------------------|
| Born before collapse, lived in city | 50+ comfort |
| Lived in survivor camp with power | 40+ comfort |
| Always lived rough | 15+ comfort |

**Effects if expectations not met:**
- Contentment drops -3 to -10 per day (based on gap)
- They may complain, refuse to work, or leave
- "High maintenance" characters leave if comfort < 30

## UNLOCKING BLUEPRINTS

### How to Find New Tech

1. **Scavenging Runs**
   - Higher danger zones = better blueprints
   - Libraries, schools, homes, stores

2. **Survivor Recruitment**
   - New survivors bring skills
   - Some have rare skills from professions

3. **Exploration**
   - Abandoned military bases = weapons
   - Universities = science/tech
   - Farms = agriculture

4. **Experimentation**
   - Combine materials to discover recipes
   - Failed experiments still teach
