# OUTPOST ZERO - Cultural Diversity System

## GOAL: 80% American, 20% International

**Rationale:** The collapse happened fast - people were on vacation, business trips, studying abroad, visiting family. Not everyone made it home.

---

## CHARACTER DISTRIBUTION

### Target: 50-100 starting survivors

| Nationality | Count | % | Background |
|-------------|-------|---|------------|
| American | 40-80 | 80% | Various states, backgrounds |
| Chinese | 4-8 | 8-10% | Students, workers, tourists |
| Korean | 2-4 | 4-5% | Students, workers |
| Japanese | 2-4 | 4-5% | Tourists, business |
| Mexican | 2-4 | 4-5% | Various |
| Other | 2-4 | 4-5% | Indian, Vietnamese, etc |

## CULTURAL CONFLICT SYSTEMS

### Language Barriers

| Language | Can Speak English | Notes |
|----------|-------------------|-------|
| American | Native | Main language |
| Chinese (Mandarin) | 50% chance | Some speak English |
| Korean | 60% chance | Many learn English |
| Japanese | 40% chance | Less common English |
| Spanish | 80% chance | Many Americans speak some |

**Effects:**
- Characters without shared language: -25% work efficiency when paired
- Misunderstandings cause arguments
- Need translator character (English + other)

### Cultural Communication Styles

| Culture | Style | Effect |
|---------|-------|--------|
| American | Direct, casual | May seem rude to others |
| Chinese | Indirect, face-saving | May seem evasive |
| Japanese | Formal, hierarchical | Hard to approach |
| Korean | Direct but warm | Similar to American |
| Mexican | Warm, relational | Good for morale |

### Death & Burial Customs

| Culture | Custom | Effect |
|---------|--------|--------|
| American | Funeral, cremation, burial | Grief ritual |
| Chinese | Ancestor worship, burning paper | +5 contentment if allowed |
| Japanese | Buddhist ceremony, cremation | Quiet respect needed |
| Korean | Ancestor rites | Family focus |
| Mexican | Dia de los Muertos celebration | Festive memorial |
| Muslim | Islamic burial (within 24h) | Quick burial needed |

**Game Effect:**
- If customs respected: +10 mental health for that character
- If customs ignored: -15 mental health
- Creates mission opportunities (find specific items)

### Religious Practices

| Religion | Practices | Needs |
|----------|----------|-------|
| Christianity | Prayer, church | Place to pray |
| Judaism | Sabbath (Shabbat) | No work from sunset Fri-Sat |
| Islam | Prayer 5x/day, Ramadan | Prayer space, halal food |
| Buddhism | Meditation, vegetarian option | Quiet space |
| Shinto | Respect for ancestors | Shrine space |
| Folk/Traditional | Varies | Ancestor items |
| Atheist | None | - |

**Game Effect:**
- Characters with religious needs: -10 contentment if not met
- Can create religious items (cross, mezuzah, prayer mat, etc.)
- Some religions have dietary restrictions (halal, kosher, vegetarian)

## CULTURAL EVENT EXAMPLES

### Language Misunderstanding
```
Event: Chinese character tries to explain something
Trigger: Paired with American on task
Outcome: -10 work efficiency, argument possible
Resolution: Learn each other's language (skill check)
```

### Burial Custom Conflict
```
Event: Chinese character wants to burn paper money for ancestors
Trigger: Has materials
Outcome: Americans find it strange, may start argument
Resolution: Build small shrine area (respect)
```

### Food Culture Clash
```
Event: Japanese character brought to kitchen
Trigger: Sees American cooking
Outcome: "That's not how you..." vs "It's just food..."
Resolution: Cultural exchange (both learn, +contentment)
```

### Religious Tension
```
Event: Muslim character needs prayer time
Trigger: During important task
Outcome: Conflict over priorities
Resolution: Schedule prayer into work (respect)
```

### Cultural Celebration
```
Event: Dia de los Muertos
Trigger: Mexican character wants to celebrate
Outcome: Can honor dead, share culture
Reward: +15 contentment for participants, new traditions
```

## GAMEPLAY BENEFITS

**Why include diverse cultures:**

1. **Rich Storytelling** - Different perspectives on survival
2. **Conflict** - Misunderstandings create drama
3. **Learning** - Player learns about other cultures
4. **Unique Skills** - Each culture brings different knowledge
5. **Representation** - More players see themselves
