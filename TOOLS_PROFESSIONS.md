## PROFESSION RESOURCES

### FARMER

**Needs from others:**
- Basic Tools (hoes, rakes) → Blacksmith/Carpenter
- Seeds → Gatherers/Scavengers
- Water → Water Bearer
- Fertilizer → Leatherworker (bones)

**Produces for others:**
- Food → Cook
- Straw → Carpenter (roofing)
- Bones → Leatherworker
- Feed → Animal Husbandry

**Tool Progression:**
- Hands (base): Slow
- Basic Rake/Hoe: +25% speed
- Quality Tools: +50% speed
- Magic/Advanced: +75% speed

## IMPLEMENTATION: TOOL CHECKS

```python
def do_farming(survivor, game):
    tool_bonus = 0
    
    # Check for tools
    has_hoe = check_inventory("Quality Hoe")
    has_rake = check_inventory("Quality Rake")
    
    if has_hoe: tool_bonus += 25
    if has_rake: tool_bonus += 25
    
    # Apply to skill
    effective_skill = survivor.skills["Farming"] + tool_bonus
    
    # Result
    roll = effective_skill + random.randint(1, 100)
```
